# 점심 메뉴 추천 앱 기획서

## 1. 프로젝트 개요
- **프로젝트명**: 오늘 뭐 먹지? (WhatToEat)
- **한줄 요약**: 직장인을 위한 AI 기반 점심 메뉴 추천 및 주변 식당 예약 서비스
- **유형**: 신규 개발
- **예상 기간**: 4개월

## 2. 문제 정의 및 해결책
### Pain Point
1. 매일 반복되는 "오늘 뭐 먹지?" 고민
2. 동료들과 메뉴 합의에 소요되는 시간 낭비
3. 주변 맛집 정보 파편화

### 해결책
- AI 기반 개인화 메뉴 추천 알고리즘
- 그룹 투표 기능으로 빠른 합의
- 실시간 대기시간 연동 예약 시스템

## 3. 타겟 사용자
- **주 타겟**: 도심 오피스 직장인 (25-45세)
- **부 타겟**: 소규모 팀 리더, 점심 약속이 잦은 영업직

## 4. 핵심 기능
1. **AI 메뉴 추천**: 날씨, 기분, 과거 이력 기반 개인화 추천
2. **그룹 투표**: 3초 만에 팀원 의견 수렴
3. **원클릭 예약**: 카카오톡/네이버 연동 간편 예약
4. **리워드 시스템**: 리뷰 작성 시 포인트 적립

## 5. 비즈니스 모델
- **식당 광고료**: CPC 기반 상위 노출
- **예약 수수료**: 예약 건당 500원
- **프리미엄 구독**: 월 3,900원 (광고 제거 + 추가 기능)
