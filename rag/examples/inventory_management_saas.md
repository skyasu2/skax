# 소상공인 재고관리 SaaS 기획서

## 1. 프로젝트 개요
- **프로젝트명**: 스톡이지 (StockEasy)
- **한줄 요약**: AI 수요 예측 기반 소상공인 맞춤형 재고관리 및 발주 자동화 SaaS
- **유형**: 신규 개발
- **예상 기간**: 4개월

## 2. 문제 정의 및 해결책
### Pain Point
1. 엑셀 기반 수동 재고 관리로 인한 오류 및 시간 낭비
2. 재고 과잉/품절로 인한 기회비용 손실
3. 발주 타이밍 및 수량 결정의 어려움

### 해결책
- 바코드/QR 스캔 기반 실시간 재고 자동 추적
- AI 수요 예측으로 최적 재고 수준 제안
- 원클릭 자동 발주 시스템

## 3. 타겟 사용자
- **주 타겟**: 동네 마트, 편의점, 소형 카페 운영자
- **부 타겟**: 온라인 쇼핑몰, 소규모 제조업체

## 4. 핵심 기능
1. **실시간 재고 현황**: 바코드 스캔 + POS 연동 자동 차감
2. **AI 수요 예측**: 계절/요일/이벤트 기반 판매량 예측
3. **스마트 발주**: 예측 기반 최적 발주량 추천 + 원클릭 발주
4. **유통기한 관리**: 임박 상품 알림 및 할인 전략 제안
5. **리포트**: 매출/재고회전율/손실률 대시보드

## 5. 비즈니스 모델
- **SaaS 구독**: 
  - Basic: 월 19,900원 (1개 매장, 기본 기능)
  - Pro: 월 39,900원 (3개 매장 + AI 예측)
  - Enterprise: 월 99,900원 (무제한 매장 + API 연동)
- **도매업체 연결 수수료**: 발주 금액의 1%
- **프리미엄 분석 리포트**: 월 9,900원 (심층 경영 분석)

## 6. 기술 스택
- **프론트엔드**: Next.js (웹), React Native (모바일)
- **백엔드**: Node.js, Redis, PostgreSQL
- **AI/ML**: Prophet (시계열 예측), scikit-learn
- **인프라**: AWS Lambda (서버리스), S3, CloudWatch

## 7. 성공 지표
- MAU 1,000개 매장 달성 (서비스 론칭 6개월 내)
- 고객 재고 손실률 30% 감소
- NPS 50 이상 유지
